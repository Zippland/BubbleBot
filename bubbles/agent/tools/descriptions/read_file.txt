Reads a file from the filesystem. Automatically handles both files and images.

**Supported file types**:
- **files**: Returns content with line numbers (cat -n format)
- **Image files** (.jpg, .jpeg, .png, .gif, .bmp, .webp, .tiff): Returns image content that you can directly see and analyze

**Path handling**:
- `~` refers to session_dir (NOT the system $HOME)
- Relative paths are resolved relative to session_dir
- All paths must be within session_dir; accessing files outside will return an error

**Usage**:
- For text files: optionally specify start_line/end_line for large files
- For images: just provide the path, the image will be displayed for you to analyze
- Images saved from chat messages are stored in `data/` directory (e.g., `data/image.jpg`)
- You can call multiple tools in a single response to read multiple files at once
