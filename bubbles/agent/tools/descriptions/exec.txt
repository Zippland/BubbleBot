Execute a shell command and return its output.

**Safety limits**:
- Commands timeout after 60 seconds (configurable)
- Dangerous commands are blocked (rm -rf, format, dd, shutdown, etc.)
- Output is truncated at 10,000 characters

**Path handling**:
- `~` in working_dir refers to session_dir (NOT the system $HOME)
- Relative working_dir is resolved relative to session_dir
- working_dir must be within session_dir; commands outside will fail
